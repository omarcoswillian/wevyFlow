Wevyflow

Exportar frame como template editável

  1. Selecione um Frame na tela do Figma
  2. Clique em Exportar frame selecionado
  3. Baixe o arquivo .wevyflow.json
  4. No WevyFlow → Criativos → Biblioteca → "Importar do Figma"
Fundo: PNG perfeito do Figma — foto, vetores, sombras, blur e efeitos, tudo incluso.
Textos: exportados como camadas editáveis separadas.
